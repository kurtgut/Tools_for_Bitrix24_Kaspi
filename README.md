Программы инструменты для работы с Битрикс 24 и Каспи
Программа 1: 
для проверки, добавлено ли нужное кастомное поле в CRM.
• 	Подключается к Bitrix24 через вебхук.
• 	Получает список всех пользовательских полей сделки.
• 	Проверяет, существует ли конкретное поле с заданным кодом. 
• 	Выводит результат проверки в консоль.

Программа 2: 
получает все данные конкретной сделки по её ID
• 	Подключается к Bitrix24 через вебхук.
• 	Запрашивает данные сделки по её ID.
• 	Выводит все поля сделки в консоль или сообщает об ошибке.

Программа 3: 
предназначена для получения полной информации о заказе в системе Kaspi.kz по его уникальному номеру (order_code). Она подключается к API Kaspi, извлекает данные и выводит как общую информацию о заказе, так и полный JSON‑ответ.

• 	Подключается к API Kaspi через токен.
• 	Получает заказ по его номеру.
• 	Выводит ключевую информацию (код, статус, сумма, клиент, доставка, оплата).
• 	Показывает полный JSON‑ответ для анализа.
